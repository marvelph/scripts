# scripts

運用対象は次の 4 スクリプトです。

- `aws-login`: AWS SSO ログイン用コマンド
- `aws-switch.sh`: `AWS_PROFILE` 切り替え用シェル関数
- `ssh-select`: `~/.ssh/config` の `Host` から接続先選択
- `ssh-add-select`: `~/.ssh/config` の `IdentityFile` から鍵を選択して `ssh-add`
